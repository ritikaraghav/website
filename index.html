This is all about getting started with Git.

It is basically a backup plan control which records changes to your file.Simply,Git is a Distributed version system(this means not only up to your pc or laptop but can be accessed from anywhere on internet) which is good enough to restore your files if you lost them and it also contains latest snapshot of your file.


The machenism that git uses for this check unique name of changes called as SHA-1 hash.This is 40 character string compose of hexadecimal characters from (0-9) and (A-F).
There are three states in Git:
1)Working Directory
2)Staging Area
3)Git Directory/Repository
